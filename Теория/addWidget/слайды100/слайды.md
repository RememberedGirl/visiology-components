# Требования к визуализации: Силовой граф

## 1. Библиотека визуализации
ECharts

## 2. Тип визуализации
Force-directed Graph (Силовой граф)

## 3. Желаемый результат

### Визуальные характеристики:
- Узлы в виде кругов с подписями
- Связи в виде линий между узлами
- Цвет узлов определяется Измерение[0] из палитры Visiology
- Размер узлов пропорционален Показатель[1] с нормализацией от 5px до 100px
- Толщина связей пропорциональна Показатель[0] с нормализацией от 1px до 9px

### Функциональность:
- Перетаскивание узлов мышью
- Приближение/отдаление графа
- При наведении на узел подсвечиваются связанные узлы и связи
- Всплывающая подсказка с информацией об узле и связях
- Анимация сил при изменении данных

## 4. Структура данных

### Измерения:
- **Измерение[0]:** Узел источника
- **Измерение[1]:** Узел назначения

### Показатели:
- **Показатель[0]:** Сила связи (для толщины линии)
- **Показатель[1]:** Вес узла (для размера круга)


## Структура данных в Visiology:
- **Измерения (Dimensions)**: находятся в `item.keys` и `item.formattedKeys` (текстовые категории)
- **Метры/показатели (Measures)**: находятся в `item.values` и `item.formattedValues` (числовые значения)
- **Названия колонок**: в `item.cols` (первые N элементов - измерения, остальные - метры)

### Пример структуры данных:
```javascript
{
    items: [
    {   
        keys: [value_dim0, value_dim1, ...], 
        values: [metric0_value, metric1_value, ...], 
        cols: ["nameimension0", "nameDimension1", ..., "nameMetric0", "nameMetric1", ...]
    }
  ]
}
```
