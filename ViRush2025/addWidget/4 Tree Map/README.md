# Требования к визуализации: Tree Map

## 1. Библиотека визуализации
ECharts

## 2. Тип визуализации
Tree Map (древовидная карта)

## 3. Желаемый результат

### Визуальные характеристики:
- Прямоугольные элементы, вложенные в родительские контейнеры
- Каждое уникальное значение Измерение[0] получает уникальный цвет из палитры Visiology
- Размер элемента пропорционален значению показателя
- Подписи внутри элементов с названием и значением
- Иерархическая структура отображается через вложенность
- Всплывающая подсказка при наведении с детальной информацией

## 4. Структура данных

### Измерения:
- **Измерение[0]:** Родительская категория (уровень 1) 
- **Измерение[1]:** Дочерняя категория (уровень 2) 
- **Измерение[2]:** Элемент (уровень 3) 

### Показатели:
- **Показатель[0]:** Значение для размера элемента 



## Контекст системы Visiology

### Основные компоненты объекта `w`:

```javascript
w = {
    // 1. ОБЩИЕ НАСТРОЙКИ ВИДЖЕТА
    "general": {
        "renderTo": "уникальный_идентификатор", // ID контейнера в DOM
    },

    // 2. ЦВЕТОВАЯ ПАЛИТРА VISIOLOGY
    "colors": [
        "rgba(40, 238, 150, 1)",
        "rgba(255, 217, 0, 1)",
        // ... массив цветов из палитры виджета
    ],

    // 3. ДАННЫЕ
    "data": {
        "primaryData": {
            "items": [ // ОСНОВНОЙ МАССИВ ДАННЫХ
                {
                    // Измерения (категории)
                    "keys": ["значение_измерения_0", "значение_измерения_1"],
                    "formattedKeys": ["форматированное_значение_0", "форматированное_значение_1"],

                    // Показатели/меры (числовые значения)
                    "values": [значение_показателя_0, значение_показателя_1],
                    "formattedValues": ["форматированное_значение_0", "форматированное_значение_1"],

                    // Названия колонок (порядок: измерения + показатели)
                    "cols": ["измерение_0", "измерение_1", "показатель_0", "показатель_1"],
                }
            ]
        }
    }
}
```
